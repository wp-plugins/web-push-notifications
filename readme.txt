== Installation ==

1. Go to WordPress Control Panel
1. Click "Plugins", then "Add New"
1. Type "Web Push Notifications" and click "Search Plugins"
1. Download and install the Plugin
1. Click the "Activate Plugin" link
1. Move to the "Notification" section in the menu
1. Done! Now you can start collect your data
